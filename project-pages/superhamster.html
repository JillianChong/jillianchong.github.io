Super Hamster Party
Overview
Super Hamster Party is a four-player, turn-based board game, where players need to successfully complete minigames or use purchased power-ups to move forward in the game. The goal of the project is to reach the end of the board before the other players. This project was built as final project for my class about functional programming. Over the span of a semester, I collaborated with three classmates (Ireanne Cao, Lina Hao, and Elaine Wu) to build a fully-functioning game. My responsibilities focused on building the graphical user interface for the game, implementing player-board and player-player interactions, and creating the hang-man minigame. As a programmer, the main objective of this project was to apply the principles of higher-order and modular programming using OCaml.
Built with: OCaml
