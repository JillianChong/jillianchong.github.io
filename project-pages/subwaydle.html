Subwaydle Unlimited
Overview
Subwaydle Unlimited is a game that combines the concept of Wordle and the routes of the MTA, or New York City public tranportation system. Following the concept of the game Subwaydle, the goal of the game is to guess the set of three trains that connect the randomly-generated start and end stations. Unlike the original Subwaydle, you aren't limited to just one puzzle a day - you can solve an unlimited number of puzzles each day.
Built with:
  • HTML, CSS, Javascript for the front-end
  • Java for the back-end
  • Python for web scraping and data processing
  • Spring Boot for merging front-end and back-end
