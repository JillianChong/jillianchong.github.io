To-Do List
Overview
To-Do List is a static interface for organizing tasks. The goal of the project is allow the user to keep track of their daily tasks by storing important information, like due date, priority level, and other task information. The interface also allows the user to mark tasks complete and easily see how many tasks are left. As a programmer, the main objective of this project was to understand the foundations of front-end development.
Built with: HTML, CSS, and Javascript
